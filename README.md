# Marketing-recommendations-for-TV-products
为了提高传统协同过滤模型的推荐精度，我们提出了一种将时间权重和兴趣相似度相结合的协同过滤模型。 时间权重考虑了用户的观看偏好变化对时间的依赖性，并且兴趣的相似性用于提高模型的预测准确性。

### 项目背景
伴随着互联网技术的快速发展和应用拓展，“三网融合”（因特网、电信网、广播电视刚）为传统广播电视媒介带来了发展机遇，广播电视运营商可以与众多的家庭用户实现信息的实时交互，使得全方位、个性化的产品营销和有偿服务成为现实
广电网络运营公司现已建设大数据基础営销服务平台，本次科创项目拟基于该平台提供的用户收视信息与产品信息，根据用户的观看记录，分析得出用户的收视喜好，并预则生成每个用户的个性推荐表。

### 研究方案

本项目研究方案主要包括以下2个方面：

1. 基于所给数据，先进行数据预处理。再按不同的电视产品（如：电视节目、电视广告等)类型、节目时长、观看次数、用户观看时间间隔等角度分析收视规律。
2. 基于分析得到的用户收视规律，得出每个用户的用户家庭成员结构与收视偏好。基于这些数据根据每个用户的偏好，预测出最可能感兴趣的节目，并生成推荐列表。

整个过程中所使用到Python与 Mariadb。
运用传统协同过算法与改进型时间可加权协同过滤算法（ NTWCF）生成用户偏好推荐模型，并通过多种模型输入参数对比预测结果，保证预测结果的准确性。
